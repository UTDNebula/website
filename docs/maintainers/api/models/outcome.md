## Overview

An `Outcome` represents a [Requirement](./requirement.md) outcome from [Exam](./exam.md) completion.

## Properties

| Name            | Type                              | Description                                            | Example |
| --------------- | --------------------------------- | ------------------------------------------------------ | ------- |
| **requirement** | [Requirement](./requirement.md)   | Requirement associated with the outcome                | N/A     |
| **outcome**     | Array - Array<ObjectId \| Credit> | Credits earned for the [Requirement](./requirement.md) | N/A     |
